Nightly Builds

If you like to use the latest development version or need a bugfix which is not yet released, you can install EGit from the following update sites:

  • EGit development build (master branch) - https://download.eclipse.org/egit/updates-nightly
  • EGit development build (master branch) - p2 repository ZIP download - https://repo.eclipse.org/content/groups/snapshots/org/eclipse/egit/org.eclipse.egit.repository/
  • EGit maintenance build (latest stable branch) - https://download.eclipse.org/egit/updates-stable-nightly

Because every change is reviewed before it is included in the nightly builds, they are actually quite stable.

You can also build EGit from scratch following our contributor guide.
